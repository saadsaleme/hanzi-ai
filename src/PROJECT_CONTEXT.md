# HanZi AI Project Context

## Project Name
HanZi AI

## Project Type
Chinese learning website for HSK 1–4.

## Main File
src/HanZiApp.jsx

## Tech Stack
React + Vite

## Main Features
- Home page
- Vocabulary section
- Flashcards
- Reading section
- Listening section
- Exercises section
- Grammar section
- Exam section
- AI Tutor
- Dashboard
- Settings
- Login / Sign up
- Subscription plans
- Free trial

## Current Goal
Improve the website and fix bugs without breaking existing features.

## Important Rules
- Do not delete existing features.
- Do not rewrite the whole project unless asked.
- Always check src/HanZiApp.jsx before editing.
- Keep the HanZi AI design style.
- Make changes step by step.
- Explain what files were changed.

## Current Known Issues
- AI Tutor still needs to be smarter and more accurate.
- Login/sign up needs stable account saving.
- Email verification needs to work.
- Duplicate emails should be blocked.
- Free trial AI Tutor should be limited to 10 messages per day.
- Vocabulary meanings must match HSK books.
- Reading/listening/exercises need more real HSK content.
- Appearance settings should apply to the whole platform.
- Language settings should translate the whole platform.

## Subscription Rules
- Standard plan: learning features without AI Tutor.
- Premium plan: includes AI Tutor.
- Free Trial: 7 days, AI Tutor limited to 10 messages/day.

## AI Tutor Rules
- Must behave like ChatGPT but only for Chinese learning.
- Must remember conversation until user clicks New Chat.
- Must answer follow-up questions using context.
- Must not repeat old examples.
- Must check verified HSK content first.

## Authentication Rules
- User account must be saved after sign up.
- User can log out and log back in.
- Same email cannot create two accounts.
- Add date of birth.
- Add captcha.
- Add email verification.
